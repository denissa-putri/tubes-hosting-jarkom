Layer 2

Switch

Switch adalah perangkat jaringan yang bekerja pada lapisan Data Link (Layer 2) pada model OSI dan berfungsi untuk menghubungkan beberapa komputer dalam suatu jaringan lokal.

Fungsi

Penghubung End-Device

Contoh

Cisco SF200-48

Referensi:Madcoms 2010: "Sistem Jaringan Komputer untuk Pemula".

Layer 3

Router

Router berfungsi untuk mengirimkan paket data dari satu network ke network yang lain sekaligus menentukan jalur terbaik (best path) untuk mencapai netwrork tujuan.

Fungsi

Menentukan jalur terbaik antar Jaringan

Contoh

MikroTik, Cisco ISR 4000

Referensi: Mikrotik Kung Fu: Kitab 4.

Wireless

Access Point

Access point adalah perangkat jaringan yang berfungsi sebagai penghubung antara jaringan kabel dengan perangkat nirkabel.

Fungsi

Menyediakan akses Wi-Fi dan memperluas jangkauan jaringan.

Contoh

Ubiquiti UniFi

Referensi: Madcoms 2010 :"Sistem Jaringan Komputer untuk Pemula"
